# Operations Guide

## Daily queue

1. Review submitted requests.
2. Move valid requests to **Under review**.
3. Confirm item details, access and contact consent.
4. Approve the request.
5. Create an assignment and schedule.
6. Confirm collection completion.
7. Add the request to a recycler handover batch.
8. Record verified weight and receipt information.
9. Complete the request after reconciliation.

## Status integrity

Never mark a request as handed to the recycler without an actual batch record. Never publish estimated weight as verified weight.

## Privacy

Do not export addresses or contact information to personal devices. Use the platform and approved communication channels.
