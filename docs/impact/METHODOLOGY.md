# Impact Measurement Methodology

## Current public metrics

The seed command creates editable records for e-waste, paper, plastic/glass, households and collections. These are starting values and are not permanently embedded in frontend code.

## Evidence hierarchy

1. certified recycler receipt or weight record;
2. EcoRevive collection manifest reconciled to a handover;
3. item-level estimate clearly labelled as estimated;
4. self-reported figure, used only with an explicit qualifier.

## Update procedure

1. reconcile completed requests with the recycler batch;
2. record verified weight;
3. document the receipt/reference;
4. update public metrics through the admin interface;
5. retain the calculation worksheet or evidence location;
6. record who approved the update.

## Environmental conversions

Do not publish carbon, landfill or resource-savings conversions until a defensible factor, source, unit and uncertainty are documented. Never present estimated environmental equivalents as recycler-verified measurements.
